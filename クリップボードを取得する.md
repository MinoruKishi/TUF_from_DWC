# すぐ使えるチートシート（最小）

  * 文字出力（確実）：`'H' emit` ／ ヌル終端文字列：`addr ztype`【】

  * 1秒待つ：`1000 ms`【】

  * ループ：`10 for 42 emit next`（`*` を10個）【】

  * メモリ文字列を出す最小例：
    
        mem 120000 + dup 'H' swap c! 1+ dup 'i' swap c! 1+ 0 swap c!
    mem 120000 + ztype
